### linux 网络设备有一种IPIP的设置, calico利用这个设备，实现ipip。

```
# ip link

16: tunl0@NONE: <NOARP,UP,LOWER_UP> mtu 1440 qdisc noqueue state UNKNOWN mode DEFAULT group default qlen 1000
    link/ipip 0.0.0.0 brd 0.0.0.0
```
